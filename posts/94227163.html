谷歌云 (GCP) 永久免费 VPS 申请指南

发表于2026-05-24|更新于2026-05-30|企业服务云服务器
|总字数:1.2k|阅读时长:3分钟|浏览量:

Google Cloud Platform (GCP) 的 Always Free 计划是目前市面上最稳定的免费 VPS 方案之一。

只要配置得当，你可以永久免费使用一台 e2-micro 配置的云服务器。但 GCP 的计费规则复杂，稍有不慎（如选错网络层级、IP 未释放）就会产生额外费用。

本文将手把手带你完成避坑配置，并教你如何设置“扣费红线”。

前提条件：
你需要一张支持外币支付的信用卡（Visa/MasterCard）用于验证身份。
注册后，必须升级为付费账号（Upgrade Billing Account） 才能解锁完整的 Compute Engine 功能。
但这并不意味着一定会扣费，只要我们在免费额度内使用即可。

第一步：创建实例 (核心避坑参数一览表)

登录 Google Cloud Console，在左侧导航栏找到 Compute Engine -> 虚拟机实例，点击 创建实例。

为了确保“零费用”，请在创建页面的各功能模块中，严格对照以下表格进行配置：

GCP 实例创建模块 核心配置项 必选参数 避坑防扣费说明
机器配置 区域 us-west1 (俄勒冈)
us-central1 (爱荷华)
us-east1 (南卡罗来纳)		 只能三选一，选择美国这三个以外的任何其他区域均会直接计费。
机器配置 机器类型 系列选择 E2
型号选择 e2-micro		 包含 2 个 vCPU 和 1 GB 内存。仅此型号属于 Always Free 免费项目。
操作系统和存储空间 启动磁盘 类型：标准永久性磁盘
大小：最大 30 GB		 点击“更改”按钮。默认通常是平衡或 SSD 磁盘，必须手动改回标准磁盘，否则产生存储费。
数据保护 备份 关闭 快照时间表，选择 无备份 快照功能默认或误开启后，会根据快照存储容量独立扣费。
网络 防火墙 勾选 允许 HTTP 流量
勾选 允许 HTTPS 流量		 用于开启常规的 80 和 443 端口，方便后续搭建公网服务，此项本身免费。
网络 网络接口 -> IP 栈类型 保持默认选择 IPv4 (单栈) 保持默认单栈即可，避免因复杂的多栈网络引起额外的流量路由误计费。
网络 网络接口 -> 外部 IPv4 地址 保持默认选择 临时 千万不要申请静态 IP！静态 IP 一旦绑定的实例关机或被删，会因“闲置”被收取高昂的闲置费。临时 IP 在实例运行期间完全免费，实例删除时也会被一起释放。
网络 网络接口 -> 网络服务层级 从 高级 切换为 标准 最大的隐藏收费陷阱！ 系统默认勾选“高级 (Premium)”。必须手动改成标准层级，才可享受每月 200 GB 的免费出站流量。
可观察性 Ops Agent 取消勾选 安装 Ops Agent 强烈建议关闭。不仅会占用 e2-micro 极其宝贵的 1G 内存，还会因为上传监控日志产生潜在的日志存储费。
安全/高级 其他自动化设置 保持默认 / 无需额外配置 请勿在高级选项内随意挂载额外的独立硬盘或配置备用网络接口，以免超出免费额度。

注意：
此时观测右侧的 每月估算费用，会显示 US$7.31 的月度费用。
细项包括：

  • 2 个 vCPU + 1 GB 内存，每月估算费用 US$6.11
  • 30 GB 标准永久性磁盘，每月估算费用 US$1.20

这些费用都是 GCP 默认提供的 Always Free 项目，不会产生实际扣费的。

第二步：设置预算报警 (最后一道防线)

为了防止因黑客攻击、流量超标或误操作导致信用卡被刷爆，我们需要设置一个极其敏感的预算提醒。

  1. 点击左侧导航栏的 Billing (结算)。
  2. 在菜单中选择 Budgets & alerts (预算和提醒)，点击 Create budget (创建预算)。
  3. 设置范围与金额：
    • 名称：随便填写，比如 月度预算。
    • 时间范围：月度 (Monthly)。
    • 金额类型：指定金额 (Specified amount)。
    • 目标金额：建议设置为 0.1 (美元)。
  4. 设置触发触发器 (Actions)：
    • 勾选 向结算管理员和用户发送邮件提醒。
    • 当实际费用或预测费用达到 50%、90%、100% 时立即触发通知。

🎉 配置完成
至此，你已经安全地拥有了一台永久免费的 Google Cloud 服务器。
如果在未来的某天，你的邮箱中收到了超过 $0.01 的账单提醒，请立即登录后台检查是否误开启了自动快照、选错了高级网络层级，或是创建了额外的未挂载磁盘。
