本文详细介绍如何自建高速、稳定的代理节点，涵盖多种流行协议及管理面板。
无论你是个人用户还是小型团队，本文都能帮你搭建稳定、高速的代理网络！

Shadowsocks 协议

这是一个简单的对称加密算法，对请求加密后，是无规则字节流。
特征明显，容易被 GFW 检测到，会被封端口甚至拉黑 IP。

ss-server: https://github.com/shadowsocks/shadowsocks-libev
ss-client: https://github.com/shadowsocks/shadowsocks-windows
v2ray-plugin: https://github.com/shadowsocks/v2ray-plugin

通过 apt 直接搭建服务：

修改配置文件 /etc/shadowsocks-libev/config.json 后重启服务。
server 改成允许所有ip访问，server_port 服务端口；
method 加密方法，password 加密密码；

这时，客户端 ss-client 填写 服务器ip、服务端口8388、method、password 就可以翻墙了。

可以通过安装 v2ray-plugin 对请求进行伪装，降低被探测的风险（被封的几率还是很大）。

服务器端：apt install shadowsocks-v2ray-plugin

trojan 协议不对数据进行加密，会对请求添加协议头，表明这是 trojan 协议，以及添加一个密码（用来验证）。
然后交给 TLS 进行完整加密。
服务端利用 TLS 解密后，检测是否是有效的 trojan 协议，如果不是，就会反代到配置的伪装网站。

VMess 协议

VMess 的加密方式比较复杂，一般选择 auto，会随机推选一种加密方式和加密密码，对原请求进行加密；
然后将 加密方式 和 加密密码 ，记录到 vmess 的协议头，利用 用户id 对此部位进行加密；
再基于 时间戳浮动 和 用户id，进行 md5哈希 计算，写到协议头，这是 vmess 检测请求是否有效的方法（生成时间范围的大数组，检验是否在数组内）。

还有一个额外id，比如设置 1，是说再额外利用一个用户id进行哈希计算，因为流量太多了不够用。
新版本都是设置 额外id 为 0，表示利用 aead 加密，因为哈希不安全。

一键脚本：https://github.com/v2fly/fhs-install-v2ray

这样加密的数据，也是无规则的字节流，和 Shadowsocks 协议一样，很容易被探测到。
利用 v2ray 强大的功能，可以引入 TLS 加密，以及用 websocket 协议承载（后续会解释为什么加一层 ws）。

直接修改配置文件：

vim /etc/systemd/system/v2ray.service
把 User=nobody 删除掉，再启动服务就可以了。

客户端这时，只需把 传输层协议 改成 ws，传输层安全 改成 tls 就可以了。

外面还可以再套一层 web，将 TLS 协议交给 nginx 解析。
nginx 增加反代配置，将路径 /vmess 请求交给 vmess 服务，其他请求转发到伪装网站。

客户端，只需在 传输层协议路径，配置好路径 /vmess 就可以了。

这就是 VMess + WS + TLS + WEB。

VLess 协议

VLess 协议，类似于 Trojan 协议，也是不对数据进行加密，交给 TLS 加密。
它的优势是，协议头比较精简。

一键脚本：https://github.com/XTLS/Xray-install

利用一键脚本安装服务：

编辑配置文件 /usr/local/etc/xray/config.json

这时启动会报错，是由于服务权限不足，无法读取证书文件。
vim /etc/systemd/system/xray.service
把 User=nobody 删除掉，再启动服务。

这是搭建了 vless + tcp + tls 的服务，然后利用了 xray 的回落功能，
当请求的协议不是 vless 时，将请求交给 trojan 服务，这个也配置了回落到 伪装网站。

客户端，就可以配置使用 vless 节点 或者 trojan 节点了。（入口端口是 2433，tcp + tls）

面板服务本身，是通过 http 访问的，提示不安全，建议升级到 https。
这里就不利用面板提供的一键功能了，不想暴露端口到公网。
通过 nginx 做一层 SSL 443 端口的反代到面板服务就可以。
然后在 【面板设置 -> 常规 -> 证书】，【面板监听IP】为 127.0.0.1 只允许本机。
例如，反代域名是 myxui.131499.xyz，只需访问 https://myxui.131499.xyz/VjiPY18FZc0HLY5 就可以登录面板了。
