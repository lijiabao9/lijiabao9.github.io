Hexo 博客搭建教程

发表于2025-09-22|更新于2025-10-25|Hexo博客
|总字数:1.8k|阅读时长:7分钟|浏览量:

Hexo 是一款快速、简洁且高效的博客框架。
本文记录了 Hexo 的搭建过程，使用最新版本的 hexo 8.0.0（2025-09-16 Released）。

官网文档请看 文档 | Hexo

环境要求

  • git

    • 版本无要求
    • 实际版本：git version 2.51.0.windows.1

  • NodeJS

    • Hexo 7.0+，要求 NodeJS >= 14.0.0
    • 实际版本：v24.8.0

直接全局安装 hexo，方便使用。

命令 hexo -v 有输出，代表安装成功。

一键部署到GitHub-Pages

本地启动，只是为了看效果。最终博客还是需要部署到公网的。

GitHub 提供了免费的 GitHub Pages，可进行网站托管。

  1. 注册 GitHub 账号，比如我的用户名是 lijiabao9

  2. 然后创建一个 public 的仓库，仓库名称为 lijiabao9.github.io （格式有要求，username.github.io）

  3. 本地生成 SSH 公私钥，将公钥配置到 GitHub，才可以操作远程仓库

    命令行执行 ssh-keygen -t rsa -C "Your mail@example.com，
    连续三次回车后，在 ~/.ssh/ 下，会生成公私钥文件 id_rsa，id_rsa.pub ，
    文本编辑器打开公钥文件 id_rsa.pub，复制内容。

    GitHub 主页，Setting -> SSH and GPG keys -> SSH keys -> new SSH key，
    Title 填写 Acer（这个可以任意填写，建议填写主机名称，方便区分）；
    Key type 保持默认的 Authentication Key；
    Key 粘贴复制的公钥内容；
    点击 Add SSH key 新增保存。

  4. 配置文件 _config.yaml，增加仓库信息

  5. 安装部署插件 hexo-deployer-git

  6. 通过命令 hexo d （deploy）一键部署

    这个命令，本质是先 clean 然后 generate，生成了 /public/ 静态资源文件；
    将 /public/ 静态资源文件，上传到指定的 GitHub 仓库上；
    由于仓库是 GitHub Pages，会将默认分支 main 进行部署（可在仓库的 Settings -> Pages -> Build and deployment，选择从 main 分支部署）；
    等待部署完成后，访问 https://lijiabao9.github.io 就是博客网站了；

自动推送部署

由于后续项目体积会变大，每次执行 hexo d 命令的时间会变长。

考虑将 hexo-blog 项目源码托管到 GitHub 上，给项目设置 GitHub Action，检测到分支代码有变更，自动打包推送到 Pages 仓库。

  1. 在 GitHub 上新建一个 private 仓库，命名为 hexo-blog（可随意命名，就是一个普通的仓库）

  2. 需要给 GitHub Action 提供一个 token，否则推送没权限

    GitHub 主页，Settings -> Developer Settings -> Personal access tokens -> Token(classic) -> Generate new token(classic)

    Note 填写 ‘Hexo 自动部署 Token’（这是备注可任意填写）；
    Expiration 选择 No expiration 永不过期；
    select scopes 勾选 repo 和 workflow；

    点击 Generate token 后，会将 token 只显示一次，复制保存下来。

  3. 在本地 hexo-blog 项目，新建文件 /.github/workflows/autodeploy.yml，这是 action 工作流配置

    记得替换配置的 Pages 仓库信息，以及 token。

  4. 将本地仓库，关联到远程仓库

    后面就只需要维护管理源码就可以了，打包部署工作自动完成。

SEO优化插件(建议)

hexo-abbrlink：将文章链接转为数字字母

hexo-filter-nofollow：给所有超链接元素的 rel 属性，加上 nofollow 等值

hexo-generator-feed：生成 atom.xml 文件，方便搜索引擎收录

hexo-generator-sitemap：生成 sitemap.xml 文件，方便搜索引擎收录
