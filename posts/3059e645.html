SSH远程连接服务器

默认情况下，首次安装 Ubuntu 时，不允许通过 SSH 进行远程访问。

  1. 安装 SSH 服务
  2. 端口（默认22）放行
    Ubuntu 自带一个配置防火墙配置工具，称为 UFW。
    如果防火墙在你的系统上被启用，请确保放行了 SSH 端口或关闭防火墙（不推荐）：

  3. 修改登录连接配置（默认只允许通过密钥登录）
    编辑 ssh 配置文件 vi /etc/ssh/sshd_config，
    修改 PasswordAuthentication 为 yes （允许通过密码连接）
    修改 PermitRootLogin 为 yes （允许用 root 用户连接）

    如果是 Oracle Cloud 实例，还需要修改配置文件 vi /etc/ssh/sshd_config.d/60-cloudimg-settings.conf
    修改 PasswordAuthentication 为 yes （允许通过密码连接）

    然后重启 ssh 服务

新增/删除用户：
创建用户有两条命令：adduer 和 useradd，
对应着两条删除用户的命令：deluser 和 userdel；
这两种命令之间的区别：
adduser（推荐这种前缀指令）：会自动为创建的用户指定主目录、系统shell版本，会在创建时输入用户密码。
useradd：需要使用参数选项指定上述基本设置，如果不使用任何参数，则创建的用户无密码、无主目录、没有指定shell版本。
