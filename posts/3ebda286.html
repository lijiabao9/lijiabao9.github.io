Gotify 自托管消息推送服务部署指南

发表于2026-06-02|更新于2026-06-18|服务部署
|总字数:490|阅读时长:2分钟|浏览量:

Gotify 是免费、开源、完全自托管的消息推送服务，包含 Web 前端和 WebSocket 后端，提供 Android 客户端，可实现秒级消息推送。

适用于系统告警、脚本执行结果、定时任务状态等通知场景。

官方网站 | GitHub 仓库

2. Nginx 反向代理与 HTTPS

Android 客户端需 HTTPS 连接，建议配置 Nginx 反代。

3. 使用指南

创建应用

  1. 访问 https://gotify.example.com 登录（默认 admin/admin）
  2. 点击 APPS → Create Application
  3. 输入应用名称，创建后获取 Token

发送消息

参数 说明
title 消息标题
message 消息正文（支持 Markdown）
priority 优先级（0-10，数值越大提示越明显）

4. Android 客户端配置

  1. 下载 Gotify Android APK
  2. 输入服务器地址 https://gotify.example.com，点击 Check URL
  3. 输入账号密码登录（建议单独创建普通用户，避免使用管理员账号）
  4. 输入客户端名称（手机标识），点击 Create
  5. 允许系统通知，并设置 允许后台运行 / 忽略电池优化
