Google Workspace 土耳其区超低价注册 .com 域名

发表于2026-06-15|更新于2026-06-30|域名管理
|总字数:1.4k|阅读时长:4分钟|浏览量:

一般来说，.com 域名的常规注册价格在 70 CNY / 年 左右，续费更是居高不下。但通过 Google Workspace 土耳其区（TRY里拉 计价），可以做到：

  • 首年： ₺18.76 TRY ≈ 3 元
  • 续费： ₺75 TRY ≈ 12 元 / 年

本文记录完整操作流程，包含账号注册、风控应对与订阅取消等关键细节。

准备工作

项目 说明
Google 账号 已有账号（用于接收验证邮件）
VISA 信用卡 支持国际支付，用于付款
手机号 支持 +86 接收验证码（仅风控验证用）
土耳其地址 可使用在线地址生成器

1. 注册 Google Workspace

进入注册流程

访问 Google Workspace 官网，点击 开始免费使用 → 创建新账号。

填写企业信息

字段 推荐填写
公司名称 任意，如 MyHomeLab
员工人数 只有您一人
区域 土耳其（关键！利用土区价格）

点击 下一步。

联系信息

字段 说明
姓氏/名字 任意
当前电子邮件地址 你的原有 Google 账号（后续接收域名验证邮件）

点击 下一步。

选择域名

选择 获取新的自定义域名 → 点击 继续。

在搜索框输入想要的域名，系统会显示价格：

首年约 3 元，续费约 12 元。确认后点击目标域名。

输入企业信息（土耳其地址）

保持勾选 将我的联系信息设为不公开。

使用 土耳其地址生成器 生成以下内容：

  • 街道地址
  • 邮政编码
  • 区 / 省
  • 电话号码（需 +90 开头，此号码不会验证）

创建登录凭证

字段 说明
用户名 任意（如 admin）
密码 自定义登录密码

点击 同意并继续。

2. 账号验证与风控

页面将跳转至新账号登录页：

  • 用户名：admin@example.com
  • 密码：刚刚设置的密码

首次登录可能触发风控：

  • 需要手机号接收短信验证码
  • 可使用 +86 手机号（仅用于此次验证，不会绑定账号）
  • 输入验证码后完成登录验证

✅ 账号创建完成！

3. 试用与付款

跳过方案选择

系统会弹出 商务标准版（₺170.80 TRY/月）的试用弹窗。

点击 试用 14 天，后续可取消。

填写付款信息

在结账页面：

项目 操作
商务标准版 ₺170.80 TRY/月（试用期结束后可取消）
域名 ₺75.00 TRY/年（首年 ₺18.76）
付款方式 输入 VISA 卡号、有效期、CVC
邮寄地址 默认同土耳其地址
域名隐私保护 保持勾选

点击 同意并继续。

⚠️ 常见风控：若出现付款验证提示，需上传身份证/护照等证件验证。正常通过则跳过此步。

跳过用户添加

添加用户页面点击 暂时跳过。

个性化设置页面均选择 我不确定。

4. 激活域名

完成注册后，原有的 Google 账号 会收到两封邮件：

  1. 域名购买成功邮件
    You’ve successfully purchased example.com

  2. Squarespace 验证邮件（Google 通过 Squarespace 注册域名）
    Domains Pending Verification

点击第二封邮件中的 VERIFY NOW 按钮，以激活域名。

进入 Squarespace 登录页，选择 Continue with Google，使用新账号 admin@example.com 登录。

登录后即可看到已注册的域名 example.com。

5. DNS 托管（可选）

如希望将域名托管到 Cloudflare：

  1. Squarespace 控制台左侧 → DNS → DNS Nameservers
  2. 选择 Custom nameservers
  3. 填入 Cloudflare 分配的地址
  4. 确认关闭 DNSSEC（按提示操作）

6. 取消 Google Workspace 订阅（重要）

⚠️ Workspace 订阅与域名订阅相互独立，取消前者不会影响域名所有权。

登录控制台

访问 Google 管理控制台，使用 admin@example.com 登录。

取消商务标准版订阅

左侧导航栏 → 结算 → 订阅 → 点击 Google Workspace 商务标准版 → 更多 → 取消订阅。

选择任意理由 → 确认 → 输入当前账号邮箱 → 取消我的订阅。

此计划的权益包括，2TB 的谷歌云盘
取消此计划后，账号将无法使用 GMail 等服务

查看剩余订阅

取消后，订阅列表仅剩 域名注册 一项

无法一次性支付多年费用，也无法提前续订，系统会在到期时自动续订一年。
需要确保信用卡可正常扣费。

费用总结

项目 首年 续费（每年）
.com 域名 ₺18.76 TRY（≈ ¥3） ₺75 TRY（≈ ¥12）
Google Workspace 14 天试用，可随时取消 已取消，无需付费

汇率仅供参考（1 TRY ≈ 0.16 CNY），实际以信用卡账单为准。

注意事项

  1. 区域选择不可更改：注册时必须选择土耳其，后续无法修改
  2. 信用卡风控：部分国内 VISA 卡可能被拒，可尝试更换卡片
  3. 域名隐私保护：默认开启，无需额外付费
  4. Workspace 试用到期：14 天后自动转为付费，建议提前取消
  5. 域名所有权：取消 Workspace 后域名仍归你所有，可正常转移或续费

常见问题

Q：取消 Workspace 后 Gmail 还能用吗？
A：不能。取消后 admin@example.com 的 Gmail 服务将停止，但域名不受影响。

Q：可以把域名转移到其他注册商吗？
A：可以。Squarespace 支持域名转移，需获取转移码并解锁域名。

Q：续费价格会变动吗？
A：Google 和 Squarespace 保留调整价格的权利，建议关注官方通知。

Q：注册时提示域名不可用？
A：换一个未被注册的 .com 域名即可。
