Docker Desktop

镜像容器管理工具。

需要系统开启 WSL 功能，参考我的文章：WSL2安装开启。

下载安装

下载地址：https://www.docker.com/products/docker-desktop/

双击下载的 exe 引导程序，完成安装：

默认是安装在 C 盘，且安装过程不提供自定义配置的。

强制安装在 D 盘

首先在 D 盘创建 2 个目录：

  • D:\Program Files\Docker：Docker Desktop 的安装目录。

  • D:\Program Files\Docker\data：Docker 用于存储镜像、容器、卷等数据的目录

打开命令提示符（管理员），进入到上述下载的 exe 引导程序所在目录，然后执行命令：

命令行末换行 \ 是为了方便展示，执行有问题的话，合并成一行命令执行。

参数解释：

  • --installation-dir="D:\Program Files\Docker"：指定 Docker Desktop 的安装目录。

  • --wsl-default-data-root="D:\Program Files\Docker\data"：指定 WSL 存储 Docker 数据的目录。

  • --windows-containers-default-data-root="D:\\Program Files\\Docker"：注意双斜杠，用于 Windows 容器存储路径。

执行命令后，会弹出之前的安装界面，等待安装完成即可。
