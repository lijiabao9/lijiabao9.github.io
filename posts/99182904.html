Artalk 自托管评论系统部署与高级功能定制指南

发表于2026-05-25|更新于2026-06-16|服务部署
|总字数:1.1k|阅读时长:5分钟|浏览量:

Artalk 是一款简洁、轻量且功能强大的自托管评论系统，提供完善的管理后台和数据隐私掌控。

本文通过 Docker Compose 部署服务端并进行深度定制。

官方网站 | GitHub 仓库

2. 界面个性化

后台 设置 → 界面配置：

配置项 推荐值
评论框占位文字 让思绪在此停泊，或如风，或如诗…
无评论显示文字 此处静默，却等待你的第一笔心动。
发送按钮文字 寄往光的深处
表情包 https://cdn.jsdelivr.net/gh/eryajf/emotion_generate/dist/artalk.json
投票按钮 勾选
反对按钮 不勾选
用户UA徽标 勾选

设置 → IP属地：勾选 开启 IP 属地展示

6. 社交登录

设置 → 社交登录：

  • 勾选 启用社交登录
  • 建议同时开启 允许匿名评论
