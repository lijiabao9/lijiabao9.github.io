二、初始化配置与使用

1. 修改默认密码

现在你可以通过 http://ip:8080 访问管理后台。

  • 默认账号：admin
  • 默认密码：umami

⚠️ 重要：登录后，请立即点击左下角用户名 -> Settings -> Profile -> Change password 修改你的初始密码。

2. 添加网站与引入脚本

  1. 在 Websites 页面，点击 Add website。
  2. 填入你的 网站名称 和 网站网址。

  3. 保存后，点击列表中的 Edit 按钮，你会得到

    • Website ID：这是唯一标识你的网站。
