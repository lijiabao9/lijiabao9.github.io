Butterfly 是 Hexo 生态中广受欢迎的高颜值主题，以其强大的功能和极高的可定制性著称。

本文将系统记录我如何将 Butterfly 主题应用到极致，打造一个集美观与交互于一体的个人博客。

2. 页面构建与个性化配置

2.1 创建内置标准页面

Butterfly 主题的诸多特殊页面依赖特定的 Front-matter 属性。

使用 hexo new page "页面名" 创建以下核心页面，并修改对应的 Markdown 文件。

具体参数可参考 官方页面配置文档。

  • 标签页 (tags)：配置 type: "tags"
  • 分类页 (categories)：配置 type: "categories"
  • 友情链接 (link)：配置 type: "link"（需配合数据文件 source/_data/link.yml 使用）
  • 说说/朋友圈 (shuoshuo)：配置 type: "shuoshuo"（需配合数据文件 source/_data/shuoshuo.yml 使用）
  • 404 页面：在主题配置文件中开启 error_404: true 即可自动化生成。

2.2 主题个性化设置

参考 官方主题配置文档。

其余细节及更多动效，可根据需要参考官方文档逐步解锁。
