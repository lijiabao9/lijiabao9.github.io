2. 页面构建与个性化配置

2.1 创建内置标准页面

Butterfly 的诸多特殊页面依赖特定的 Front-matter 属性。使用 hexo new page 创建以下页面：

  • 标签页 (tags) — 配置 type: "tags"
  • 分类页 (categories) — 配置 type: "categories"
  • 友情链接 (link) — 配置 type: "link"，需配合 source/_data/link.yml 使用
  • 说说 (shuoshuo) — 配置 type: "shuoshuo"，需配合 source/_data/shuoshuo.yml 使用
  • 404 页面 — 在主题配置中开启 error_404: true 即可自动生成

2.2 主题个性化设置

参考官方配置文档 进行个性化调整，其余细节及动效根据需求逐步解锁。
