Markdown 现代核心语法全景指南

发表于2026-05-23|更新于2026-05-24|编程语言DevOps
|总字数:1.3k|阅读时长:4分钟|浏览量:

Markdown 是一种极轻量级的文本标记语言，它允许创作者使用易读、易写的纯文本格式编写文档，通过极简的标记符号实现全自动的结构化排版。

掌握 Markdown，能让你彻底摆脱频繁调整鼠标字号、行距的繁琐操作，将全部精力无缝聚焦于内容创作本身。

1. 标题语法

要创建标题，在单词或短语前面添加井号（#），并紧跟一个空格。

# 号的数量代表了标题的级别，Markdown 支持 1~6 级标题。

语法效果
Markdown 语法效果预览
# 一级标题一级标题
## 二级标题二级标题
### 三级标题三级标题
#### 四级标题四级标题
##### 五级标题五级标题
###### 六级标题六级标题

2. 强调语法

Markdown 使用星号 * 或下划线 _ 来标记需要强调的文本。

  • 要用 斜体 显示文本，在单词或短语前后添加一个星号 *。
  • 要 加粗 文本，在单词或短语的前后各添加两个星号 **。
  • 要同时用 粗体和斜体 突出显示文本，在单词或短语的前后各添加三个星号 ***。
语法效果

Italicized text is the *cat’s meow*.Italicized text is the cat’s meow.
I just love **bold text**.I just love bold text.
This text is ***really important***.This text is really important.

3. 引用语法

要创建块引用（Blockquotes），在段落前添加一个 > 符号，后面紧跟一个空格。引用块可以嵌套，也可以包含其他 Markdown 元素。

语法效果

渲染效果如下所示：

I’ve got my heart set on reaching out to the world.

I’m not in the mood to go back to the beginning.

4. 列表语法

可以将多个条目组织成有序或无序列表。

  • 有序列表：在每个列表项前添加数字，并紧跟一个英文句点，再紧跟一个空格。数字不必按数学顺序排列，但列表将以数字 1 起始。
  • 无序列表：在每个列表项前面添加破折号 -、星号 * 或加号 +，后面紧跟一个空格。
有序列表

渲染效果如下所示：

  2. Second item
  3. Third item

    2. Indented item
  4. Fourth item
无序列表

渲染效果如下所示：

  • First item
  • Second item

    • Indented item
    • Indented item
  • Fourth item

5. 代码语法

  • 行内代码：要将单词或短语表示为代码，将其包裹在反引号 (`) 中。
  • 代码块：在代码块前后行上使用三个反引号（```）。在首行的反引号后指定语言名称（如 json, python, java），可实现语法高亮。

注意：
如果 行内代码 包含行内代码，最外层请使用双反引号 (``) 包裹；
如果 代码块 中包含代码块，最外层请使用四个反引号（````）包裹。

6. 分隔线语法

要创建分隔线，在单独一行上使用三个或多个破折号 (---)、星号 (***) 或下划线 (___) ，并且行内不能包含其他内容。

7. 链接语法

链接文本放在 方括号 内，链接地址放在后面的 圆括号 中，链接 title（鼠标悬停提示）为可选内容。

语法格式：[content](url title)

9. 转义字符语法

如果要显示 Markdown 语法中使用的特殊字符（如 *, #, \ 等），需在字符前面添加反斜杠 \ 进行转义。

语法效果

10. 表格语法

使用三个或多个连字符（---）创建表头分隔线，并使用管道符（|）分隔每一列。

对齐方式：

  • :--- 左对齐
  • :---: 居中对齐
  • ---: 右对齐

11. 删除线语法

要给文本添加删除线，请使用两个波浪号 ~~ 包裹文本。
