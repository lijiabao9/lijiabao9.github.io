Windows Terminal

Win11 默认安装了 Windows Terminal （终端管理器）应用。

win + r 输入 wt 可快速打开应用。

默认集成了：Prompt Command（命令提示符）、PowerShell（5.1）以及 Azure Cloud Shell。

通过命令一键安装（不推荐）：

如果下载很慢，则是网络问题。使用上面的 ps7 终端，已经做了代理。

这里有个天坑，字体安装，会将字体安装到用户级别下（~\AppData\Local\Microsoft\Windows\Fonts），
这里的文件是无法直接删除的，被核心进程 vcshost.exe 持有，此进程无法结束。

win + R 输入 regedit 打开注册表，地址输入：
计算机\HKEY_CURRENT_USER\Software\Microsoft\Windows NT\CurrentVersion\Fonts
删除对应的注册表项 Oh-My-Posh，删除此注册表项目后，就可以手动删除上述的字体目录了。

推荐直接下载字体文件：https://lijiabao.lanzoul.com/iIkbj36pmegh
将所有的 .tiff 字体文件，拖拽到 C:\WINDOWS\Fonts\ 目录下，为全局安装字体。
