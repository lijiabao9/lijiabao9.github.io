发表于2024-04-27|更新于2024-06-08|应用搭建

ChatGPT 使用需要科学上网，并且中国大陆是被禁止注册的，需要使用外国的手机号，才可注册。

本文介绍，怎样在自己的电脑上，搭建一个大模型服务。

使用效果类似 ChatGPT。

Ollama安装

ollama 是一个旨在简化大型语言模型本地部署和运行过程的工具。它提供了一个轻量级、易于扩展的框架，让开发者能够在本地机器上轻松构建和管理LLMs。通过 ollama，开发者可以访问和运行一系列预构建的模型，或者导入和定制自己的模型，无需关注复杂的底层实现细节。

Ollama官网

Ollama Github 主页

下载链接

运行下载下来的引导程序 OllamaSetup.exe，安装 Ollama。

这种运行单个大模型，而且只能在命令窗口交互的方式，不是很优雅。

下面介绍一种更优雅的方式。

安装docker

Docker官网

点击 Download for Windows，按照提示安装即可。

此时，在命令窗口，检验是否安装成功。

由于docker的官方库，位于国外，下载特别慢，需要配置镜像。

使用阿里云镜像源，需要个人注册登录，搜索 容器镜像服务，

在 镜像工具 > 镜像加速器，生成个人的加速器地址，复制下来。

然后在 docker 桌面应用，右上角的 齿轮 设置，选择 Docker Engine

填写

配置了阿里云镜像加速后，docker 拉取 iamge 时，就不要开代理了。

安装OpenWebUI

Open WebUI 是一个可扩展、功能丰富且用户友好的自托管 WebUI，旨在完全离线操作。 它支持各种 LLM 运行程序，包括 Ollama 和 OpenAI 兼容的 API。
