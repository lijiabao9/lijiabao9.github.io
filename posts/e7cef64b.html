3. 核心效率工具

Zoxide（智能目录跳转）

记录访问过的目录，支持模糊匹配跳转。

  • Bash：eval "$(zoxide init bash --cmd cd)"
  • Fish：zoxide init fish --cmd cd | source

Fzf（模糊搜索）

通用命令行模糊搜索器，可检索历史命令、文件及目录。

  • Bash：eval "$(fzf --bash)"
  • Fish：fzf --fish | source

快捷键：

快捷键 功能
Ctrl + R 历史命令搜索
Ctrl + T 文件搜索
Alt + C 目录搜索并进入

Atuin（历史记录管理）

使用 SQLite 数据库替代普通历史文件，支持跨设备同步。

  • Bash：eval "$(atuin init bash --disable-up-arrow)"
  • Fish：atuin init fish --disable-up-arrow | source

Too Long; Didn’t Read，提供命令核心用法及高频示例，替代冗长 man 手册。
