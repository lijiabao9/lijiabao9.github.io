免费获取 Microsoft 365 E3 企业开发者订阅与 5TB OneDrive 扩容全攻略

发表于2026-05-23|更新于2026-05-25|资源分享白嫖教程
|总字数:1.6k|阅读时长:5分钟|浏览量:

Microsoft 365 E3 订阅通常是企业级的付费服务，但通过特定的开发者计划或 Symphony 试用链接，我们可以免费获取。

核心权益：

  • Office 365 全家桶：支持激活本地桌面版 Word, Excel, PPT 等。
  • 25 个许可证：可以创建 25 个子账号，分发给亲友使用。
  • 5TB OneDrive：每个账号均可享受最高 5TB 的云存储空间。
  • 自动续期：保持账号活跃（如 API 调用），微软通常会按年自动续期。

免责声明：
本教程利用的是微软公开的试用/开发者链接（如 Symphony/Dev 渠道）。此类订阅存在不确定性，微软随时可能收回权益或封禁账号。
请勿在 OneDrive 中存储没有任何备份的重要数据！建议仅作为从属/备用网盘或用于激活 Office 软件。

第一步：创建空全局管理员

我们需要先注册一个没有绑定任何订阅的“空壳”微软企业账号。

  1. 访问注册链接：
    点击此处跳转注册页面

  2. 验证邮箱：
    输入任意电子邮件，点击 下一步，如果提示账号不存在，点击 创建账号。

  3. 填写信息：

    • 基本情况：随意填写，点击 下一步。
    • 登录方法：设置你的全局管理员前缀（如 admin）和 **专用域名（xxx.onmicrosoft.com）。
    • 重要：请务必牢记此处的用户名和密码，这是你的最高权限账号。

  4. 跳过无关项：

    • 地址：随意填写（提示无法认证也不用管）。
    • 纳税人识别号：随意填写一个身份证号。
    • 关键点：当流程进行到 付款 或 添加支付方式 页面时，直接关闭网页，不要填写。

此时，你已经拥有了一个空的全局管理员账号：admin@xxxx.onmicrosoft.com。

第二步：激活 E3 订阅

利用特定的订阅链接，将 E3 订阅挂载到刚才创建的账号上。

  1. 访问购买链接：
    点击此处跳转 E3 Dev 订阅下单页
    (如果提示登录，请使用第一步创建的管理员账号登录)

  2. 配置支付方式：
    在购物车页面，按照当前系统提示绑定您的信用卡（如果系统仍能刷出支付宝，可优先选择支付宝）。

    ⚠️ 2026年5月最新变动与重要警告（风控升级）：

    • 历史情况：在 2025 年 12 月期间，用户在结账时可顺利将付款方式切换为 Alipay (支付宝)，无需验证，直接无感白嫖。

    • 当前实测：系统会强制要求绑定信用卡（Credit Card）进行身份核验。

    • 关于费用：虽然下单页面显示的订单总计依然为 0 元，但信用卡验证阶段可能会有扣准授权或扣除少量验证金（随后退还）的动作。

    • 操作建议：如果您没有闲置的测试信用卡，或出于隐私安全考虑不愿绑定真实卡片，建议在此步骤谨慎操作。若坚持激活，可尝试使用国内双币信用卡或国外虚拟卡进行绑卡测试。

  3. 完成下单：
    点击右侧的 Place order。当看到成功提示，即表示订阅挂载成功！

  4. 验证许可证：

    • 登录 Microsoft 365 Admin Center
    • 左侧导航栏，进入 Billing (账单) → Your products (你的产品)
    • 可以看到有 Office 365 E3 Developer 订阅已激活，包含 25 个许可证

  5. 自动按年度续费：

    • 点击 Office 365 E3 Develop 后的三个点 → Edit recurring billing (编辑定期账单)，改为 Renew（Renew yearly） 按年度自动续订
    • 注意：点击 Edit billing frequency (编辑计费频率)，切换按月/按年后，订阅有效期会自动顺延

进阶玩法 (备用链接)：

如果不满足于 25 个许可证，同一账号可以重复购买叠加。

还可以尝试以下 200 万个许可证 的 Symphony 订阅（同一账号订阅类型会互斥，只能选一种）：

  • E1 Symphony (后续可在产品中升级为 E3): 点击跳转
  • E3 Symphony (推荐): 点击跳转

第三步：扩容 OneDrive 至 5TB

365 默认为每一个用户分配 1TB 的 OneDrive 空间，修改设置，将默认分配的 OneDrive 空间设为最大的 5TB。

  1. 在 365 Admin Center 左侧导航栏，点击 SharePoint

  2. 进入到 SharePoint Admin Center 后，左侧导航栏，点击 Settings → OneDrive → Storage limit

  3. 将默认存储限制 Default storage limit，修改为 5120 (GB)，点击保存

第四步：创建子用户 (降低风险)

强烈建议不要直接使用全局管理员账号进行日常使用，以降低风控概率。请创建子用户用于激活 Office 和存储数据。

  1. 回到 365 Admin Center，点击 Users → Active users → Add a user 添加用户

    • Basic：填写 Display name（用户名）、Username（账号），勾选自动生成密码，或者手动填写密码
    • Product licenses：location 默认保持 China，勾选分配 Office 365 E3 Developer 许可证
    • Optional settings：Roles 角色配置，保持默认的 User (no admin center access) 作为普通用户
    • 创建成功后，复制保存账号和密码，后续用于子账号的登录

  2. 检查空间： 在 Active Users 界面，可以看到新建的用户，点击该用户查看详情，在 OneDrive 选项卡，可看到 “Storage used” 空间使用情况

    • 如果显示 1024GB，点击 Edit 手动改为 5120GB
    • 如果显示“未初始化”，请使用该子账号登录一次 MySignins 即可触发初始化

第五步：在 Windows 上使用子用户

现在的子账号已经是一个功能完备的“工作账号”了，可以与你原本的“个人账号”完美共存。

注意：此子账号，可以同时在 10 台设备上登录激活 Office 应用

  1. 子账号登录
    打开 Windows 的 设置 → 账户 → 登录工作或学校账号。
    点击 连接，输入子用户的账号和密码，进行登录，这将允许该账号 SSO 登录系统内的 Office 应用。

  2. 激活 Office
    打开 Word ，点击左下角的 账号 → 切换许可证 → 选择刚刚添加的 E3 子账号。

  3. 挂载 OneDrive
    右键任务栏 OneDrive 图标 → 设置
    选择 账户 → 添加账户，登录子账号后，你的文件资源管理器中会出现一个新的 OneDrive - <你的组织名> 文件夹，独享 5TB 云空间。
