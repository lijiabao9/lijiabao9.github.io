Hexo Butterfly 标签外挂完全指南 (原生 + Plus增强版)

发表于2025-11-30|更新于2025-12-01|创意工坊建站日志
|总字数:5.6k|阅读时长:24分钟|浏览量:

本文整合了 Butterfly 主题原生标签 与 Tag Plugins Plus 增强插件 的所有用法。
通过这些标签外挂，你可以轻松在 Markdown 中实现复杂的排版和交互效果。

前置说明：

  1. 带有 [原生] 标识的标签，仅需 Butterfly 主题即可使用。
  2. 带有 [Plus] 标识的标签，需要安装 hexo-butterfly-tag-plugins-plus 插件。

相关资源

  • 原生标签：Butterfly 文档(四) 标签外挂
  • Plus标签：Tag Plugins Plus | Akilarの糖果屋

1. 基础文本与排版

1.1 Label 高亮标签 [原生]

用于高亮强调文中的一段文字。

  • text: 文字内容
  • color: 背景颜色（留空 / blue / pink / red / purple / orange / green）

臣亮言：先帝创业未半，而中道崩殂。
今天下三分，益州疲敝，此诚危急存亡之秋也！
然侍卫之臣，不懈于内；忠志之士，忘身于外者。
若有作奸、犯科，及为忠善者。

1.2 行内文本样式 [Plus]

提供下划线、着重号、波浪线、删除线、键盘按键、密码隐藏等样式。

  1. 带 下划线 的文本
  2. 带 着重号 的文本
  3. 带 波浪线 的文本
  4. 带 删除线 的文本
  5. 键盘样式：Ctrl + C
  6. 密码样式：鼠标悬停查看密码

1.3 Span/P 彩色与字体控制 [Plus]

控制行内(span)或段落(p)的颜色、大小和字体。

参数(空格分隔):

  • 颜色: red, yellow, green, cyan, blue, gray
  • 大小: small, h4, h3, h2, h1, large, huge, ultra
  • 对齐: left, center, right
  • 字体: logo, code

在一段话中插入：红色、绿色、蓝色。

专为诗歌设计的居中排版。

2. 提示与警告

2.1 Note 提示块 [原生]

最常用的提示块，支持多种风格。

  • color（标识）: 留空 / primary / success / info / warning / danger
  • color（颜色）: 留空 / blue / pink / red / purple / orange / green
  • icon: 可选 fontawesome 图标（例如 'fab fa-cc-visa'），或填 no-icon
  • style: 可选 simple / modern / flat / disabled 覆盖主题的默认配置

2.2 Tip 顶部标签提示 [Plus]

样式类似 note 的渐变标签。

样式（可填）:

  • 留空 / success / info / warning / error
  • bolt / ban / home / sync / cogs / key / bell
  • 以及 font-awesome 图标，例如 fa-atom

2.3 Tip Animated 动态提示 [Plus]

为 tip 添加动画效果。
可参考 font-awesome-animation 的用法。

  • 动画类名: faa-horizontal, faa-flash, faa-spin, faa-tada 等。
  • 触发方式: animated (一直动画), animated-hover (悬停动画)。
  • 触发速度: faa-fast, faa-slow。

warning 持续水平

ban 悬停快速闪烁

3. 布局与容器

3.1 Tabs 选项卡 [原生]

用于切换显示不同的内容。

标签语法：

  • UniqueName: 唯一ID（同一个页面），不能包含空格。
  • index: 可选，默认选中的索引（从 1 开始，-1 为都不选中，不填时为第一个）。
  • @icon: 可选 fontawesome 图标（例如 '@fas fa-bomb'）

4. 按钮与链接

4.1 Btn 按钮 [原生]

单个按钮链接。

  • url: 链接地址。
  • text: 按钮文字。
  • icon: 可选 fontawesome 图标（例如 'far fa-hand-point-right'）
  • color: 背景色 或 字体边框色。（留空 / blue / pink / red / purple / orange / green）
  • style: 样式。留空（实心）/ outline（空心）
  • layout: 布局。留空（行内）/ block（块级）
  • position: 位置。留空（左对齐）/ center（居中）/ right（右对齐）
  • size: 按钮大小。留空（默认）/large（大）

5.3 Mermaid 流程图 [原生]

需要主题配置开启 mermaid: enable: true。

使用 mermaid 标签可以绘制 Flowchart（流程图）、Sequence diagram（时序图 ）、Class Diagram（类别图）、State Diagram（状态图）、Gantt（甘特图）和 Pie Chart（圆形图）。
具体语法参阅 Mermaid

    graph TD;

B-->D;

5.4 Chartjs 图标 [原生]

需要主题配置开启 chartjs: enable: true。

使用 chartjs 标签可以绘制 Line（折线图）、Bar（柱状图）、Bubble（气泡图）、Pie（饼图）等多种图形。

具体语法参阅 Chart.js

{
"type": "line",
"data": {

"datasets": [{

"borderColor": "rgb(255, 99, 132)",
"data": [0, 10, 5, 2, 20, 30, 45]
}]
},
"options": {
"responsive": true,

"display": true,

}
}

6.4 Icon 图标 [Plus]

需要引入对应 Symbol 图标 js 文件。

  • icon-xxxx: 图标名称
  • font-size: 图标大小。直接填写数字即可，单位为 em。默认值为 1em。

6.5 Bdage 徽标 [Plus]

GitHub 徽标。

参数分为三组，用 || 分隔。

  1. 基本参数
    • right: 必填参数。徽标右侧文字。
    • left: 必填参数。徽标左侧文字。
    • logo: 徽标图标。详见 simple-icons
  2. 信息参数
    • color: 徽标右边的颜色。例如 abcdef
    • link: 徽标的跳转链接。
    • title: 鼠标悬停时显示的文字（已失效）。
  3. 拓展参数
    • option: 自定义参数，支持 shields.io 的全部API参数支持

6.6 Bubble 气泡注释 [Plus]

  • content: 注释词汇
  • notation: 悬停显示的注解内容
  • background-color: 可选，气泡背景色。默认为 #71a4e3

  • id: 引用的编号。需要和引用出处的编号一致。
  • literature: 引文的内容。
  • url: 引文出处的链接。
